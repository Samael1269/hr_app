# ATS Recruitment System
A web-based Applicant Tracking System (ATS) for managing job postings, candidates, resumes, and recruitment progress.

## Features
- Job posting management
- Candidate application system
- Resume upload and storage
- Candidate detail tracking
- Application status updates
- Interview management
- AI-style match score interface

## Tech Stack
- HTML
- CSS
- JavaScript
- PHP
- MySQL
- XAMPP

## Setup
1. Place the project folder in `htdocs`
2. Start Apache and MySQL in XAMPP
3. Import the database in phpMyAdmin
4. Update `config.php`
5. Open `http://localhost/hr_app/`

## Main Tables
`users`, `jobs`, `candidates`, `applications`, `resume_files`, `interviews`
